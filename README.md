# starrocks_web
一个简单的项目，运营和监控starrocks（基于sr3.3.7版本）
包括sr的routine load,物化视图作业的查询，任务的启动和停止，界面化的操作更方便
 -routine load的基础作业信息，创建时间、作业停止时间、作业创建信息、导入数据量和错误信息等
 -物化视图的的基础作业信息、数据量
 -物化视图的依赖关系查询
